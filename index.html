Work Experience

  • ASHA (Advanced Sensing and Hand-Movement Aid)

    • Aug 2023 - Sep 2023

    • ASHA, derived from the Sanskrit word meaning 'hope,' stands for Advanced Sensing and Hand-Movement Aid, a post-stroke rehabilitation device.
    • The components include an MPU6050 Accelerometer & Gyroscope Module, a DFPlayer Mini, and a Speaker, all interfaced with an Arduino Uno and mounted on a simple glove.
    • This setup gathers analog data through the MPU6050 module, which is transmitted to the Arduino Uno.
    • It then processes this data to generate corresponding vocal responses through the speaker, using the DF Player Mini.

  • ASTA (Automated Students Tracking Application)

    • Nov 2022 - Nov 2022

    • ASTA, the Sanskrit word which means reliable is a Student Security System based on the RFID principle.
    • It focuses on foolproof surveillance of students inside the school premises.
    • The main motive behind this project is to create a parental-friendly environment for students which provides safety to them from any kind of nuisance activities and a secured journey from their respective houses to school.
    • This setup gathers analog data through the MPU6050 module, which is transmitted to the Arduino Uno.
    • We’ve used an ARDUINO UNO microcontroller board which acts as the motherboard here and a RFID RC522 Reader/Writer module that reads the UID (Unique Identification Number) of the students’, present in the ID-Card which will transmit the UID to the processor of the microcontroller board through radio waves and thereby through the signals of the microcontroller board the SIM 900A GSM Module will send a notification to the parent's mobile phone.
